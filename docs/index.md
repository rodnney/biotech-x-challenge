# Desafio DevOps Biotech-X

Bem-vindo à documentação técnica da plataforma de bioinformática da Biotech-X.

Esta solução propõe uma arquitetura **Cloud-Native** na AWS, focada em alta disponibilidade, segurança e otimização de custos para processamento de dados de espectrometria.

## Links Rápidos
* [Arquitetura Detalhada](architecture.md)
* [Runbook SRE](sre.md)

## Stack Tecnológica
* **Infraestrutura:** Terraform
* **Orquestração:** Kubernetes (EKS)
* **Computação Assíncrona:** AWS Batch + Spot Instances
* **CI/CD:** GitHub Actions
